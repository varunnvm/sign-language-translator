# 🤟 Sign Language to Speech Translator using AI
Overview:-
This project is a computer vision based application that detects sign language hand gestures and converts them into speech in real time. It helps bridge the communication gap between people who use sign language and those who do not understand it.

Features:
- Real-time hand gesture detection
- Converts gestures into text
- Speech output using Text-to-Speech
- Webcam based interaction
- Simple web interface

Technologies Used:
- Python
- OpenCV
- MediaPipe
- TensorFlow / Keras
- HTML, CSS, JavaScript
